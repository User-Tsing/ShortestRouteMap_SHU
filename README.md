SHORTEST ROUTE MAP DEMO OF SHU              
上海大学_通信与信息工程学院_数据结构与算法基础A_校园最短路径漫游_demo            
实现功能：自定义绘制地图（非API），最短路径求取，地图缩放展示，路况出行方式设置，等。           
运行方式：Qt Creator中打开.pro文件然后点击运行即可（前置：配置环境变量编译器路径等），或者建立同名项目然后把所有内容复制进去。
